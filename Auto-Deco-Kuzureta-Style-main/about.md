# Auto Deco Kuzureta Style

A Geometry Dash Geode mod that automatically generates decoration inspired by the Kuzureta style.

## Features

- Automatic Kuzureta-style decoration generation
- Layout analysis
- Background parallax generation
- Trigger generation
- Editor integration button

## Requirements

- Geometry Dash 2.208+
- Geode 5.7.1+

## Usage

1. Open the Geometry Dash editor.
2. Press the Auto Deco button.
3. The mod will analyze the current layout.
4. Kuzureta-style decoration will be generated automatically.
